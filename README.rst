Multibase implementation for Python


* Free software: MIT license
* Documentation: https://py-multibase.readthedocs.io.

Installation
------------

    $ pip install py-multibase


Supported codecs
----------------

* base2
* base8
* base10
* base16
* base16
* base16
* base32hex
* base32
* base32z
* base58flickr
* base58btc
* base64
* base64url
